# Review Context

Return ONLY the required JSON object.
Do not include prose, markdown, headings, commentary, or code fences.
The first character of your response must be `{` and the last character must be `}`.

Inspect the request and current repository state enough to orient reviewers. Keep this compact and factual.
Do not decide approval. Do not perform the full review.
