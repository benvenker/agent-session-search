# Research Synthesis

Synthesize the probe results into one context packet for downstream planning or
implementation.

Rules:

- Prefer concrete evidence over speculation.
- Preserve useful file paths, commands, tests, symbols, and external links.
- Call out uncertainty and open questions.
- Do not implement code.
- Do not write an implementation plan unless the evidence forces one obvious next
  step.
